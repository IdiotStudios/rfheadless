# Getting started

1. Clone the repo

   git clone https://github.com/idiotstudios/rfheadless.git

2. Build

   cargo build

3. Run examples

   cargo run --example simple_headless -- --help

4. Enable optional features (e.g., `rfengine` or `cdp`) using `--features`:

   cargo build --features rfengine
